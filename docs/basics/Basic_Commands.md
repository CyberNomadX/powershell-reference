## Check if file exists
To check if a file exsists use the `Test-Path` commmand.  

Example:  
`Test-Path -Path "C:\path\to\your\file.txt"`  

Return results:  
If the file exsists the return will be `$True`.  
If the file does not exsist the return will be `$False`.
